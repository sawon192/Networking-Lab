# Packet Analysis using Scapy

## Description

This project is a simple packet analyzer developed using Python and the Scapy library. It captures network packets and displays basic information such as source IP, destination IP, protocol, MAC addresses, and packet summary.

## Features

- Capture 10 network packets
- Display source and destination IP addresses
- Show TCP, UDP, and ICMP packet information
- Display source and destination MAC addresses
- Print packet summary

## Requirements

- Python 
- Scapy

## Install Scapy

```bash
pip install scapy
```

## Run the Program

```bash
python packet_analysis.py
```

## Output

The program displays packet information on the terminal after capturing network traffic.
