# Networking Lab

This repository contains my Networking Lab assignments and practice programs. The projects are developed as part of my university networking course using Python socket programming and packet analysis tools.

## Objectives

* Learn basic networking concepts
* Understand socket programming
* Build client-server applications
* Practice TCP/IP communication
* Analyze network packets

## Projects

### 1. Socket Programming

* Basic client-server communication
* TCP socket implementation
* Sending and receiving messages

### 2. Client-Server Chat

* Two-way chat application
* Multiple message exchange
* Client and server communication using sockets

### 3. Network Calculator

* Client sends mathematical expressions
* Server performs calculations
* Result is returned to the client

### 4. Packet Analysis

* Capture network packets
* Analyze packet information
* Study different network protocols

## Tools Used

* Python
* Socket Library
* Wireshark
* VS Code

## How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Run the server program first.
4. Run the client program.
5. Test the communication between client and server.

## Learning Outcomes

After completing these lab tasks, I learned:

* How socket programming works
* Communication between client and server
* Basic TCP/IP networking
* Packet capturing and analysis
* Practical implementation of networking concepts
