
# Client-Server Chat

## Objective

Build a simple chat application using TCP socket programming.

## Features

* Two-way communication
* Client and server can exchange multiple messages
* Type **bye** to exit

## Files

* client.py
* server.py

## How to Run

Run the server first.

```
python server.py
```

Run the client.

```
python client.py
```

Start chatting.
